# NTPU AI Aquila Project — Complete Website Code

這是一套可直接部署到 GitHub Pages 的完整網站程式碼。

## 檔案
- `index.html`：網站完整結構與內容
- `style.css`：網站完整樣式與動畫
- `script.js`：中英文切換、行動版選單與互動功能

## 部署方式
1. 將三個檔案與圖片資料夾上傳至 GitHub Pages repository 根目錄。
2. GitHub repository → Settings → Pages。
3. Source 選擇 `Deploy from a branch`。
4. Branch 選 `main`，資料夾選 `/root`，按 Save。

- 完整保留已完成的 Hero：AI 晶片、雙層旋轉軌道、節點、掃描線、科技網格、粒子、浮動標籤與跑馬燈。
